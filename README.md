# Fitness Score Prediction using Linear Regression 🏃‍♀️💪

## 📌 وصف المشروع
مشروع تعلم آلة يهدف إلى التنبؤ بمستوى اللياقة البدنية بناءً على عدد الخطوات اليومية باستخدام خوارزمية الانحدار الخطي (Linear Regression).

---

## 🧠 التقنيات المستخدمة
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

---

## 📊 خطوات العمل
- إنشاء مجموعة بيانات بسيطة  
- تقسيم البيانات إلى تدريب واختبار  
- بناء نموذج Linear Regression  
- تدريب النموذج  
- التنبؤ بالقيم  
- تقييم النموذج باستخدام MSE و R2 Score  
- عرض النتائج بصريًا  

---

## 📈 النتائج
تم تقييم النموذج باستخدام:
- Mean Squared Error (MSE)  
- R2 Score  
- Visualization للمقارنة بين القيم الحقيقية والمتوقعة  

---

## 🚀 الهدف من المشروع
فهم العلاقة بين النشاط البدني ومستوى اللياقة باستخدام الذكاء الاصطناعي.
